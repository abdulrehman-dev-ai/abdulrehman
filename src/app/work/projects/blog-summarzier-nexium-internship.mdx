## Overview

**Blog Summariser & Urdu Translator** is a full-stack AI-inspired web application designed to process blog URLs, extract their content, generate meaningful English summaries, and translate them into Urdu for multilingual accessibility.

Developed as part of Nexium Ltd's Summer Internship Program, this project demonstrates practical use of **natural language processing, translation APIs**, and **dual-database architecture** using **Supabase** and **MongoDB**. The result is a polished, real-time summarization tool with a beautiful and responsive interface.

## ğŸ§  Challenges & Learnings

- **Blog Text Extraction**: Some blogs resist scraping, requiring smart fallback logic.
- **Translation Reliability**: MyMemory API provides free tier translation but needs retries or throttling under load.
